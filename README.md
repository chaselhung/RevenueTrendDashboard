# 收入趋势看板（Excel数据可视化）📊

## 项目简介

该项目是一个基于 Web 的 **收入趋势分析看板**，旨在通过上传 Excel 文件，自动解析其中的客户数据并生成可视化图表📈。用户可以查看和排序客户的收入趋势，分析各项产品（如天翼、宽带、固话和 ITV）在不同月份的收入表现。此项目特别适用于财务分析、数据分析等领域的可视化展示🔍。

## 主要功能 ✨

- **数据上传与解析**：支持上传 Excel 格式文件，自动解析并展示客户数据📂  
- **数据筛选与排序**：按客户名称、经理、编码等字段进行筛选🔎，并根据总收入进行排序🔢  
- **收入趋势图表**：生成动态的月收入图表📅，展示各客户在不同月份的收入变化趋势  
- **详细客户信息**：点击任一客户，显示该客户的详细信息（收入、所属区域、产品分布等）📋  
- **响应式设计**：支持桌面端和移动端，数据和图表自动适应不同屏幕尺寸📱💻  

## 技术栈 🛠️

- **前端**：HTML, CSS, JavaScript (ES6+), ECharts（用于图表展示）📊  
- **文件处理**：SheetJS（用于解析 Excel 文件）📑  
- **响应式布局**：Flexbox📐  

## 使用方式 🚀

1. **克隆该项目**：  
   git clone https://github.com/chaselhung/RevTrendDashboard.git  

2. **打开 index.html 文件，在浏览器中查看🌍。**  

3. **上传 Excel 文件：上传 Excel 文件，自动解析并生成图表📈。**
